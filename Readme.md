# 🚀 JavaScript Problem Set

A collection of **50 JavaScript coding problems** designed to improve problem-solving skills, JavaScript fundamentals, ES6+, algorithms, and logical thinking.

## 📚 Overview

This repository contains:

- ✅ 50 JavaScript Problems
- 📅 10 Days Challenge
- 📝 5 Problems Per Day
- 🎯 Easy → Medium → Hard Difficulty
- 💡 Hints for every problem
- 🔥 Perfect for interview preparation and daily practice

---

## 🎯 Goal

Solve **5 problems every day** to:

- Strengthen JavaScript fundamentals
- Improve logical thinking
- Practice ES6+ features
- Learn problem-solving techniques
- Prepare for coding interviews

# 🚀 How to Use

1. Clone the repository.

```bash
git clone https://github.com/your-username/javascript-problem-set.git
```

2. Open the project.

```bash
cd javascript-problem-set
```

3. Solve **5 problems every day**.

4. Run your JavaScript files.

```bash
node filename.js
```
